Rules for the Game:

  1. This Game Works only on Web
  2. To start the Game press  Enter.
  3. To move the scale to right press  D.
  4. To move the scale to left press  A.
  5. The scale Which have ball you are in that scale.
  6. The target of the Game is move the ball in between the two Scales. The scale which misses to hold/catch the ball will be considered as lost.
